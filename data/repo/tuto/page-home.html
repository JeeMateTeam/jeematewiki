Page Home 42

La page Home de JeeMate permet d'ajouter tous type de widget.

Par défaut cette page est vide, pour ajouter un widget :

  1. cliquer sur le crayon
  2. cliquer sur + pour ajouter un élément à la page “Home”

Choisir le type d'élément a ajouter :

  1. Application : applications Android ou IOs
  2. Widgets : un équipement Jeedom (liste les équipements choisis dans le plugin JeeMate)
  3. Scénarios : raccourci vers un scénario JeeMate (liste les scénarios choisis dans le plugin JeeMate)
  4. Ajouter un groupe de widgets
  5. Ajouter un widget customisé

Applications 

An choisissant “Application", vous pouvez ajouter un raccourci sur une application de votre téléphone dans la page “Home”.

L'application sélectionnée apparaitra dans la page Home.

En cliquant dessus, cela ouvrira directement l'application en question.

Pour éditer (supprimer) une application, clic long sur l'icone de l'application => supprimer.

Widgets

An choisissant “Widgets", vous pouvez ajouter un ou plusieurs équipements Jeedom dans la page “Home” de JeeMate.

  1. cliquer sur l'oject/pièce pour faire apparaitre les équipements de ce dernier
  2. utiliser le sélecteur pour ajouter un ou plusieurs équipements
  3. cliquer sur le bouton sauvegarder

Rendu dans mise en forme des équipements ajoutés.

Par défaut tous les équipements sont au format 1x1.

Pour éditer (déplacer, redimensionner) la page “Home”

  1. cliquer sur le crayon
  2. utiliser le sélecteur pour passer en mode édition
  1. lors du passage en mode édition
  2. le crayon passe en mode édition

Rendu après redimensionnement et déplacement des tuiles 

  • Pour déplacer => rester appuyer sur la tuile et déplacer
  • Pour redimensionner => boucle clic sur la tuile et utiliser les flèches pour redimensionner.
